# electron-hello-world

This is an exmaple for this slide: https://hackmd.io/p/rJpxFxuu

## How?

### Clone this example
```shell
$ git clone https://github.com/YogaPan/electron-hello-world.git
```

### Install dependencies
```shell
$ npm install
```

### Start development
```shell
$ npm start
```

### Build your application
```shell
$ npm run dist
```
